# Introduction

eslint-plugin-vue-scoped-css (opens new window) is ESLint plugin for Scoped CSS in Vue.js (opens new window).

# Features

This ESLint plugin provides linting rules relate to better ways to help you avoid problems when using Scoped CSS in Vue.js (opens new window).

  • Provides linting rules for Scoped CSS.
  • Supports CSS syntax including level 4 selectors.
  • Supports <style lang="scss">.
  • Supports <style lang="stylus">.
  • Parses <style>, <template> and <script> blocks.

You can check on the Online DEMO.
